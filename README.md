# مصباح العلم — بوابة الاشتراك المدفوع

موقع يسمح للمستخدمين بالاشتراك مقابل 10$ شهرياً عبر بطاقة ائتمانية، والحصول على رمز شخصي مكوّن من 5 أرقام يمنحهم الوصول إلى أداة Gemini.

## كيف يعمل النظام
1. الزائر يضغط "الاشتراك الآن" → يُحوَّل إلى صفحة دفع آمنة من Stripe.
2. بعد الدفع، يحصل فوراً على رمز شخصي من 5 أرقام.
3. يدخل الرمز في صفحة `/access` → يُنقل تلقائياً إلى رابط أداة Gemini الحالي.
4. الرمز يبقى صالحاً طالما الاشتراك نشط. إذا فشل الدفع أو أُلغي الاشتراك، يتوقف الرمز عن العمل فوراً.
5. أنت (كمدير) تحدّث رابط أداة Gemini كل شهر من صفحة `/admin`.

## الصيانة الشهرية
كل شهر، افتح `/admin`، وألصق رابط Gemini الجديد. لا حاجة لأي تعديل آخر — رموز المشتركين تبقى كما هي طالما اشتراكهم نشط.
